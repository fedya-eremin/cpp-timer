# Time App - Таймер и Секундомер

Qt6 приложение с функциями таймера и секундомера с поддержкой светлой/тёмной темы.

## Особенности

- **Таймер**: Установка времени с часами, минутами, секундами
- **Секундомер**: Точное время с точностью до сантисекунд
- **Темы**: Переключение между светлой и тёмной темой (Ctrl+T)

## Сборка

```bash
make
```

Исполняемый файл будет создан в `bin/timeapp`.

## Запуск

```bash
./bin/timeapp
```
